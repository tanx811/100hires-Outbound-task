## Why these 10

All ten run outbound as their day job (agency, in-house quota, or training built from their own pipeline), not as theorists repackaging other people's frameworks. They cover the three channels that make up a real cold outreach pipeline — cold email (Berman, Dang, Braun), LinkedIn outreach (Reamer, Tatulea, Aitken), and cold calling (Venetz, Cegelski/Farrokh) — plus one (Dunning) who is candid about where cold outreach is *failing* in 2026, which is necessary signal for a balanced playbook.

## YouTube transcripts: collection method

Verbatim transcripts in `research/youtube-transcripts/` were pulled using the [Supadata](https://supadata.ai/) API (auto-captions). Direct YouTube caption endpoints (`timedtext` API, `yt-dlp`, `youtube-transcript-api`) all returned HTTP 429 from the build environment's shared IP and were not usable — this is noted in commit history for reproducibility. Each transcript folder also contains an earlier supplementary-notes file written before the API was available; those files are kept for context but the `*-transcript.md` files are the primary source material.
